Introducción al lenguaje

HTML es el lenguaje de marcado estándar para crear páginas web.
Permite definir la estructura y el contenido de un documento mediante etiquetas.

Etiquetas de Formato

La etiqueta "hr" se utiliza para crear una línea divisoria temática, mientras que "br"
sirve para insertar un salto de línea simple dentro de un bloque de texto.
